# 🌻 像素植物大战僵尸 (Pixel Plants vs Zombies)

一个仿照经典游戏《植物大战僵尸》制作的简化版 HTML5 像素风格游戏。

![Game Preview](./preview.png)

## 🎮 在线游玩

**直接访问：** https://laladuizhang.github.io/zwdzjs

无需安装，浏览器打开即可玩！

## ✨ 游戏特点

- 🎨 **像素风格** - 仿照经典像素游戏设计
- 🌻 **向日葵** - 定期产生阳光，维持经济
- 🫛 **豌豆射手** - 发射豌豆攻击僵尸
- 🥜 **坚果墙** - 高血量防御型植物
- 🧟 **僵尸入侵** - 从右侧不断来袭的僵尸大军

## 🕹️ 操作方式

1. 点击按钮选择要种植的植物
2. 点击草地格子种植植物
3. 点击掉落的阳光收集
4. 点击"开始游戏"启动防御战
5. 阻止僵尸到达左侧房子！

## 🌱 植物图鉴

| 植物 | 阳光消耗 | 特点 |
|------|----------|------|

## 🛠️ 技术栈

- HTML5 Canvas
- JavaScript (ES6+)
- 纯前端实现，无需后端

## 🚀 本地运行

直接在浏览器中打开 `pvz-game.html` 文件即可：

```bash
# 或者使用 Python 启动本地服务器
python -m http.server 8000
# 然后访问 http://localhost:8000/pvz-game.html
```
